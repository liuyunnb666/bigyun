# Code Navigation

- Backend entrypoints are under each service module `src/main/java` application class.
- Gateway routes are configured in Nacos templates under `BigYun-cloud/sql/nacos`.
- MyBatis XML files live under each service `src/main/resources/mapper` directory.
- Vue3 dynamic menus come from system menu SQL; static routes are in `BigYun-cloud-Vue3/src/router/index.js`.
- Provider pages are under `BigYun-cloud-Vue3/src/views/provider` and `src/views/config`.
